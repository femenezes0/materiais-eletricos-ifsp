# 17. Condutividade em Materiais Não Estequiométricos (Wüstita)

## Significado Físico

A wüstita ($Fe_{1-x}O$) possui vacâncias de ferro. Para manter a neutralidade elétrica, íons $Fe^{3+}$ são criados, atuando como lacunas ("buracos") que permitem a condução do tipo *p*.

## Fórmulas Utilizadas

[
\sigma = p|e|\mu_h
]

[
p = \frac{\text{Portadores}}{V_{\text{célula}}}
]

[
V_{\text{célula}} = a^3
]

## Dados

* (x = 0,060)
* (a = 0,437,nm = 4,37 \times 10^{-10},m)
* (\mu_h = 1,0 \times 10^{-5},m^2/V\cdot s)
* (|e| = 1,6 \times 10^{-19},C)

Na estrutura tipo NaCl existem 4 unidades de fórmula por célula unitária. Como cada vacância gera dois portadores (Fe^{3+}), existem (8x) portadores por célula.

## Resolução

### Concentração de lacunas

[
p=\frac{8x}{a^3}
]

[
p=\frac{8(0,060)}{(4,37\times10^{-10})^3}
]

[
p\approx 5,75\times10^{27};m^{-3}
]

### Condutividade

[
\sigma=(5,75\times10^{27})(1,6\times10^{-19})(1,0\times10^{-5})
]

[
\sigma\approx92,0;S/m
]

## Resultado

[
\boxed{\sigma \approx 9,20\times10^{1};S/m}
]

# 18. Alteração de Espaçamento entre Placas

## Fórmula

[
C=\frac{\varepsilon_0\varepsilon_rA}{l}
]

Como (C) e (A) permanecem constantes:

[
\frac{\varepsilon_{r1}}{l_1}=
\frac{\varepsilon_{r2}}{l_2}
]

## Dados

* (\varepsilon_{r1}=2,5)
* (l_1=1,mm)
* (\varepsilon_{r2}=4,0)

## Cálculo

[
l_2=\frac{\varepsilon_{r2}}{\varepsilon_{r1}}l_1
]

[
l_2=\frac{4,0}{2,5}(1,mm)
]

[
l_2=1,6,mm
]

## Resultado

[
\boxed{l_2 = 1,6,mm}
]

# 21. Mecanismo de Armazenamento

A inserção de um dielétrico provoca a polarização do material. O campo elétrico aplicado induz o alinhamento de dipolos, gerando um campo interno oposto que reduz a diferença de potencial para uma mesma carga. Como consequência, a capacitância aumenta e o capacitor consegue armazenar mais carga para a mesma tensão aplicada.

# 25. Tipos de Polarização

## Mecanismos

### Eletrônica

Deslocamento da nuvem eletrônica em relação ao núcleo.

### Iônica

Deslocamento relativo entre cátions e ânions.

### Orientação

Alinhamento de dipolos permanentes com o campo aplicado.

## Classificação

| Material | Tipo de Polarização             |
| -------- | ------------------------------- |
| BaTiO₃   | Eletrônica, Iônica e Orientação |

| Diamante | Eletrônica                      |
| KCl      | Eletrônica e Iônica             |

# 27. Dependência com a Frequência

Em frequências elevadas ((10^{15},Hz)), apenas a polarização eletrônica contribui para a constante dielétrica.

[
\varepsilon_r = 2,3
]

Em (1,MHz):

[
\varepsilon_{\text{total}}=6,9
]

Contribuição iônica:

[
\varepsilon_{\text{iônica}}=6,9-2,3=4,6
]

[
\frac{4,6}{6,9}\approx67%
]

## Resultado

A polarização iônica corresponde a aproximadamente **67%** da constante dielétrica total.

# 28. Temperatura de Curie

Acima da temperatura de Curie ((T_c)), a energia térmica supera as forças de alinhamento dos dipolos. O BaTiO₃ passa da estrutura tetragonal para a estrutura cúbica, tornando coincidentes os centros de carga positiva e negativa. Dessa forma, desaparece o momento dipolar permanente e o material perde o comportamento ferroelétrico.
