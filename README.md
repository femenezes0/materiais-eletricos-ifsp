# materiais-eletricos-ifsp
Repositório acadêmico desenvolvido para organização de resumos, exercícios resolvidos, perguntas e respostas e conteúdos estudados na disciplina de Materiais Elétricos no curso de Engenharia de Controle e Automação do IFSP, abordando teoria de bandas, semicondutores, propriedades elétricas dos materiais e física do estado sólido.
